# 우리의 블로그 체험단 가이드라인

## 📌 1. 체험단 가이드라인의 목적

체험단 가이드라인은 **"우리의 블로그가 추천한 저경쟁 키워드"** 를 바탕으로, 실제로 업체를 방문할 블로거들이 어떤 방식으로 글을 써야 하는지 구체적으로 안내하는 작성 설명서입니다.

### 목표

단순히 예쁘게 쓰는 리뷰가 아니라,

- ✅ **네이버 검색에 잘 노출될 수 있도록** 키워드를 전략적으로 활용하면서도
- ✅ **실제 방문 경험이 드러나는** 솔직하고 신뢰도 높은 후기를 쌓는 것

---

## 📝 2. 제목(타이틀) 작성 규칙

### 핵심 규칙

- 📌 시스템이 추천한 **저경쟁 키워드 중 최소 3개 이상을 제목에 포함**한다.
- 📌 키워드는 억지로 나열하지 말고, **자연스러운 문장 형태**로 섞어서 작성한다.
- 📌 **"누구에게 어떤 상황에 추천하고 싶은지"** 가 느껴지도록 제목을 짓는다.

### 좋은 예시

**추천 키워드:** 서현역 브런치 · 크루아상 맛집 · 분당 카페 추천

❌ **나쁜 예시** (키워드 나열식)
```
서현역 브런치 · 크루아상 맛집 · 분당 카페 추천
```

✅ **좋은 예시** (자연스러운 문장)
```
[서현역 브런치] 크루아상 맛집 찾는다면? 분당 카페 추천 여기!
```

✅ **더 좋은 예시** (상황 + 키워드)
```
데이트·주말 브런치로 딱인 서현역 크루아상 맛집
```

---

## 🔑 3. 키워드 사용 규칙 (본문)

### 핵심 규칙

- 📌 시스템이 제시한 **4개의 키워드를 각각 최소 3회 이상** 자연스럽게 언급한다.
- 📌 키워드는 **문장을 이상하게 끊어서 억지로 넣지 말고**, 실제 경험을 설명하는 흐름 안에서 녹여 쓴다.
- 📌 **같은 키워드를 연달아 반복해서 붙이지 않는다.**

### ❌ 피해야 할 사례

```
서현역 브런치 서현역 브런치 서현역 브런치 맛있었어요.
```

### ✅ 올바른 사례

```
서현역 브런치 맛집을 찾다가 들어간 이곳은 정말 마음에 들었다.
크루아상이 유명해서 크루아상을 주문했는데, 
분당 카페 중에서 이렇게 좋은 퀄리티는 처음 본다.
이제 서현역에 올 때마다 여기 브런치를 찾게 될 것 같다.
```

## 📄 4. 글 구성 & 분량

### 최소 요건

- 📌 **최소 1,000자 이상** 작성을 기본으로 한다.
- 📌 읽기 쉽게 **단락을 나누고, 소제목을 활용**한다.

### 추천하는 글 구성

```
① 방문 이유 & 위치
   - 왜 이곳을 선택했는지, 어떤 기대를 갖고 갔는지

② 매장 분위기 & 좌석
   - 인테리어, 채광, 조용한 정도, 좌석 배치, 테이블 크기 등

③ 주문 메뉴 & 맛 후기
   - 실제 주문한 메뉴, 가격, 맛, 양, 비주얼에 대한 구체적 묘사

④ 주차/대기/예약 정보
   - 도움이 되는 실제 정보 (웨이팅 시간, 주차 가능성 등)

⑤ 다시 방문 의사 & 한줄 총평
   - 솔직한 평가와 누구에게 추천하고 싶은지
```

### 반드시 포함되면 좋은 내용

✅ 방문 계기 (왜 가게 되었는지, 어떤 기대를 갖고 갔는지)

✅ 실제 주문 메뉴와 가격대 (작성 시점 기준임을 명시)

✅ 맛·양·비주얼·서비스·분위기에 대한 **구체적인 묘사**

✅ 주차, 대중교통, 도보 동선 등 **도움 되는 정보**

✅ 아쉬웠던 점이 있었다면 **솔직하지만 과장 없이 언급**

## 📸 5. 사진 & 영상 규칙

### 사진 최소 8장 이상 권장

사진은 아래 순서대로 준비하면 글의 흐름이 자연스럽습니다:

| # | 사진 종류 | 설명 |
|---|---------|------|
| 1️⃣ | 외관(간판/입구) | 업체의 첫 인상 |
| 2️⃣ | 매장 내부/좌석 분위기 | 공간의 느낌 전달 |
| 3️⃣ | 메뉴판 또는 키오스크 화면 | 가능하면 가격이 보이도록 |
| 4️⃣ | 전체 상차림 | 메뉴 구성 한눈에 보기 |
| 5️⃣ | 대표 메뉴 근접샷 | 한입샷, 단면샷 등 디테일 |
| 6️⃣ | 시그니처 포토존/인테리어 포인트 | 특징적인 장소 |
| 7️⃣ | 결제/셀프바/주방 등 특징 | 업체 특성에 맞춘 사진 |
| 8️⃣+ | 추가 사진 | 더 많을수록 좋음 |

### 영상 1개 이상 권장

- 매장 전체 분위기 영상
- 음식 나오는 장면
- '먹방 한 입' 등 자연스러운 순간
- 가로(16:9) 또는 세로(9:16) 형식 모두 가능

### 사진/영상 촬영 원칙

✅ 직접 촬영한 **원본을 사용**한다.

✅ 업체와 무관한 **타인의 얼굴은 최대한 노출하지 않는다.**

## ✅ 6. 필수 정보 항목 (체크리스트)

글을 업로드하기 전에 아래 항목들이 포함되었는지 확인하세요:

### 기본 정보
- [ ] **상호명** (정확한 이름)
- [ ] **위치** (역/출구, 근처 랜드마크)
- [ ] **도로명 주소 또는 지번 주소**

### 운영 정보
- [ ] **영업 시간**
- [ ] **브레이크 타임** (있다면)
- [ ] **라스트 오더** (알고 있는 범위에서)

### 방문 정보
- [ ] **방문 요일 / 시간대** (대기 줄 여부, 혼잡도)
- [ ] **웨이팅 방식** (번호표, 앱, 줄서기 등)

### 편의 정보
- [ ] **주차 가능 여부** (전용/공영/제휴, 비용 등)
- [ ] **주문 메뉴와 가격대** (정확한 가격은 "작성 시점 기준"임을 언급)
- [ ] **예약 가능 여부**

### 동반 가능 정보
- [ ] **아이 동반 가능 여부** (높은 의자 있는지 등)
- [ ] **반려동물 동반 가능 여부**
- [ ] **유모차 접근성**
- [ ] **휠체어 접근성**

## 💬 7. 톤 & 스타일 가이드

### 기본 톤

실제 친구에게 추천하듯 **솔직한 톤**을 권장합니다.

### 균형잡힌 표현

| 구분 | 방식 | 예시 |
|------|------|------|
| 좋았던 점 | 구체적으로 | "크루아상이 바삭하면서 버터향이 풍부했어요" |
| 아쉬운 점 | 감정을 담되 공격적이지 않게 | "가격이 좀 높은 편이라 자주 가기는 어려울 것 같아요" |
| 중립적 평가 | 객관적으로 | "매장이 좁아서 대기 시간이 상대적으로 길 수 있습니다" |

### ❌ 피해야 할 표현

- ❌ 다른 업장을 **직접적으로 비교·비하**하거나
- ❌ **타 브랜드를 깎아내리는** 표현
- ❌ 무조건적인 찬양만 나열하기
- ❌ 과도하게 부정적인 톤

### ✅ 권장하는 스타일

- ✅ 개인의 경험과 느낌을 있는 그대로
- ✅ 좋은 점과 아쉬운 점을 균형있게
- ✅ 구체적인 묘사로 신뢰도 높이기
- ✅ 누구에게 추천하고 싶은지 명확하게

## ⚠️ 8. 광고·표시 관련 필수 유의사항

### 협찬/체험 사실 명시 (필수)

글에 **반드시 아래와 같은 형태로** 명시해야 합니다:

```
해당 포스팅은 업체로부터 체험 기회를 제공받아 작성했습니다.
```

또는

```
일부 메뉴는 체험단 제공, 진솔한 개인 후기를 바탕으로 작성했습니다.
```

또는

```
[협찬] 해당 글은 체험단으로 참여하여 작성되었으며, 솔직한 의견을 담고 있습니다.
```

### 법적으로 민감한 표현 금지

❌ **의료 효능** - "피부질환 치료에 도움됩니다"
❌ **다이어트 효과** - "이 음식으로 3kg를 뺐습니다"
❌ **절대적 표현** - "절대로 실패하지 않습니다"
❌ **확정적 수식** - "완치", "100% 효과", "무조건"

### 성실한 작성의 원칙

✅ 실제 경험하지 않은 서비스/메뉴를 **경험한 것처럼 쓰지 않기**

✅ 먹지 않은 음식을 **맛본 척 쓰지 않기**

✅ 방문하지 않은 시간대에 대해 **추측으로 쓰지 않기**

## 📅 9. 게시 및 검수 가이드

### 게시 마감 기한

- 📍 **방문일 기준 3일 이내 업로드**를 권장합니다.
- 신선한 경험이 가장 생생하고 신뢰도가 높습니다.

### 임시 검수 단계 (선택)

필요한 경우, 업로드 전에 아래 절차를 거칩니다:

1. 글 작성 완료
2. **URL 또는 캡처를 "우리의 블로그" 담당자에게 제출**
3. 키워드 포함 여부, 내용 적절성 등 간단히 확인
4. 피드백 받고 수정 (필요시)
5. 최종 공개

### 수정 요청 가능성

아래 경우에는 **1회 재수정 요청**이 있을 수 있습니다:

- 🔴 **광고·법규 위반** - 명시하지 않은 협찬, 과장된 의료 표현 등
- 🔴 **키워드 누락** - 필수 키워드 4개 중 일부가 빠진 경우
- 🔴 **목적 불부합** - 가이드라인을 크게 벗어난 내용
- 🔴 **분량 미달** - 1,000자 미만

## 📋 10. 작성 전 최종 체크리스트

글을 마무리하기 전에 한 번 더 확인하세요:

### 키워드 & 제목
- [ ] 저경쟁 키워드 **3개 이상이 제목에 자연스럽게 포함**되었는가?
- [ ] **4개 키워드가 본문에 각각 3회 이상** 나타나는가?
- [ ] 제목에서 **"누구에게 추천인지"** 가 느껴지는가?

### 글 구성
- [ ] **1,000자 이상** 작성되었는가?
- [ ] **5가지 구성 요소** (방문 이유, 분위기, 메뉴, 정보, 평가)가 포함되었는가?
- [ ] 단락이 적절히 나뉘어 **읽기 쉬운가?**

### 필수 정보
- [ ] 상호명, 위치, 주소가 **정확한가?**
- [ ] 영업 시간, 가격대, 주차 정보 등 **실용적 정보가 있는가?**
- [ ] 동반 정보 (아이, 반려동물 등) **필요한 것이 포함되었는가?**

### 사진 & 영상
- [ ] **최소 8장 이상의 사진**이 있는가?
- [ ] **1개 이상의 영상**이 있는가?
- [ ] 모두 **직접 촬영한 원본**인가?

### 톤 & 표현
- [ ] **솔직하고 균형잡힌 톤**을 유지하고 있는가?
- [ ] 다른 업체 비하, 과장된 의료 표현 등 **피해야 할 표현이 없는가?**
- [ ] 아쉬운 점이 있다면 **공격적이지 않게 표현**했는가?

### 광고 표시
- [ ] **협찬/체험단 사실이 명시**되어 있는가?
- [ ] 위치는 **글 초반 또는 말미**에 눈에 띄게 있는가?
